State idea

  • state definition - class with 4 methods: preload, create, update, render and optionally init
  • only one state is active
  • gameplay of the game is to switch to state, ex. choose lang, select character form menu, fight, defeat or win screen

init method

  • uruchamia się gdy stan zostanie aktywowany - jako pierwsza metoda

preload method

  • metoda w której ściągamy assety (pliki json, pliki z ścieżkami dźwiękowymi)

create method

  • uruchamiana raz podczas tworzenie stanu
  • w niej tworzymy wszystkie obiektu występujące w tym stanie gry

update method

  • Phaser.Game#raf via requestAnimationFrame albo setTimeout
  • 38881: The core game loop - method: Phaser.Game#update

render method

  • metoda która uruchamia się po cały procesie aktualizacji obiektów w Canvas
  • w niej możemy włączyć prezentację technicznych detali naszych obiektów
  • wykonuje się zawsze po metodzie update

Day I: Here we go!

31↑

Use ECMAScript 6.
webpack i Babel.js take care for compiling to ECMAScript 5 (for browser).

webpack.config.js

                module.exports = {
                    resolve: { extensions: ['.es6.js', '.js', ''] },
                    entry: './app/scripts/main',
                    output: {
                        filename: 'bundle.js',
                        path: './app/dist'
                    },
                    module: {
                        loaders: [{
                            test: /\.es6\.js/,
                            exclude: /node_modules/,
                            loader: 'babel?stage=0'
                        }]
                    }
                }

Advantages of ECMAScript 6

  • modules - no global variables
  • importing - require only dependency
  • smooth syntax of class definition
  • "simple" inheritance
  • default parameters
  • arrow function
  • destructuring assignment
                var [x, y] = ['abc', 100]
                // x = 'abc', y = 100
